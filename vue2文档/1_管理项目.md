# 管理项目

## 运行项目

对于使用NPM的项目：

```bash
npm run serve
```

对于使用yarn的项目：

```bash
yarn run serve
```

## 安装包

对于使用NPM的项目：

```bash
npm install xxx
```

对于使用yarn的项目：

```bash
yarn add xxx
```

## 删除包

对于使用NPM的项目：

```bash
npm uninstall xxx
```

对于使用yarn的项目：

```bash
yarn remove xxx
```

## 生成

对于使用NPM的项目：

```bash
npm run build
```

对于使用yarn的项目：

```bash
yarn run build
```
